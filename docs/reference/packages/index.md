# `packages`

[README](../README.md) › `packages`

Browse repositories and install packages [[admin]]

## Subcommands

- [cancel](#command-packages-cancel) — Cancel an in-progress package installation
- [get](#command-packages-get) — Show details for one package
- [install](#command-packages-install) — Install a package
- [list](#command-packages-list) — List packages available from configured repositories
- [repos](#command-packages-repos) — Inspect or replace configured package repositories [[admin]]
  - [list](#command-packages-repos-list) — List configured package repositories
  - [set](#command-packages-repos-set) — Replace the configured package repository list

## Examples

- `jf packages list --server http://localhost:8096`
