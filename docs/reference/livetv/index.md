# `livetv`

[README](../README.md) › `livetv`

Channels, guide, recordings, timers, tuners, listings

## Subcommands

- [channels](#command-livetv-channels) — List channels
- [guide](#command-livetv-guide) — Show program guide

- [timers](#command-livetv-timers) — List timers

## Examples

- `livetv info --server http://localhost:8096`
