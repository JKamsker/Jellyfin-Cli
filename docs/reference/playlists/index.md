# `playlists`

[README](../README.md) › `playlists`

Create playlists and manage items or shares

## Subcommands

- [create](#command-playlists-create) — Create a new playlist
- [get](#command-playlists-get) — Show one playlist
- [items](#command-playlists-items) — Manage playlist items
  - [add](#command-playlists-items-add) — Add items to a playlist
  - [list](#command-playlists-items-list) — List playlist items
  - [move](#command-playlists-items-move) — Move an item within a playlist
  - [remove](#command-playlists-items-remove) — Remove items from a playlist
- [update](#command-playlists-update) — Update playlist metadata

## Examples

- `playlists create --name demo`
