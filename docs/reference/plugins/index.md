# `plugins`

[README](../README.md) › `plugins`

Configure, enable, disable, or uninstall plugins [[admin]]

## Subcommands

- [config](#command-plugins-config) — Read or update plugin configuration [[admin]]
  - [get](#command-plugins-config-get) — Read plugin configuration
  - [set](#command-plugins-config-set) — Update plugin configuration
- [disable](#command-plugins-disable) — Disable a plugin
- [enable](#command-plugins-enable) — Enable a plugin
- [list](#command-plugins-list) — List installed plugins
- [uninstall](#command-plugins-uninstall) — Uninstall a plugin

## Examples

- `plugins list --server http://localhost:8096`
