# SyncPlay

SyncPlay allows multiple users to watch content together in sync.

## List active SyncPlay groups

```bash
jf syncplay list
```

## Create a new group

```bash
jf syncplay create --name "Movie Night"
```

## Join a group

```bash
jf syncplay join --group-id <GROUP_ID>
```

## Leave the current group

```bash
jf syncplay leave
```
